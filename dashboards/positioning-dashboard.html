Supports Decision 2 · Positioning — CRM, Revenue Operations, or AI Readiness Platform?

Confidence C · Illustrative Synthetic analyst assessment Related: Positioning Canvas · Claims & Confidence

Illustrative, not validated. The per-option scores below are an initial, synthetic assessment against the Positioning Canvas. They structure the debate; they do not settle it. The evidence_strength criterion keeps aspirational framing honest — a position that sounds exciting but has no evidence is penalised.

import sys, pathlib
_root = pathlib.Path.cwd()
while _root != _root.parent and not (_root / "src").is_dir():
    _root = _root.parent
sys.path.insert(0, str(_root))

import pandas as pd
import plotly.express as px
import plotly.graph_objects as go
from IPython.display import HTML

from src.utils import load_framework, weights_from_framework
from src.scoring import weighted_score

PLOTLY_TEMPLATE = "plotly_white"
OPTION_COLORS = {
    "CRM Platform": "#6b7684",
    "Revenue Operations Platform": "#2f6bb0",
    "AI Readiness Platform": "#7a4fb0",
}
Analyst assessment of each option (synthetic, 0–10 per criterion) 
# Rows = positioning options; columns = the Positioning Canvas criteria (each 0–10).
# These are an initial analyst judgement to be revised as evidence accrues.
assessment = pd.DataFrame(
    [
        # option,                       clarity, diff, pain, investor, scale, evidence
        ["CRM Platform",                     9,    4,     7,       5,     6,     6],
        ["Revenue Operations Platform",      6,    7,     8,       7,     8,     5],
        ["AI Readiness Platform",            5,    8,     6,       9,     8,     3],
    ],
    columns=[
        "option", "clarity", "differentiation", "customer_pain_alignment",
        "investor_appeal", "scalability", "evidence_strength",
    ],
)
Apply the Positioning Canvas weights to each option 
framework = load_framework("positioning_canvas")
weights = weights_from_framework(framework)

assessment["positioning_score"] = assessment.apply(
    lambda row: weighted_score(row, weights), axis=1
)
ranked = assessment.sort_values("positioning_score", ascending=False).reset_index(drop=True)
ranked.insert(0, "rank", ranked.index + 1)
Build summary metrics (computed, not hardcoded) 
leader = ranked.iloc[0]
second = ranked.iloc[1]
margin = leader["positioning_score"] - second["positioning_score"]
weakest_evidence = assessment.sort_values("evidence_strength").iloc[0]

def metric(label, value, sub=""):
    sub_html = f'<div class="metric-sub">{sub}</div>' if sub else ""
    return (f'<div class="metric"><div class="metric-label">{label}</div>'
            f'<div class="metric-value">{value}</div>{sub_html}</div>')

HTML(
    '<div class="metric-row">'
    + metric("Options compared", len(ranked), "candidate positionings")
    + metric("Leading framing", leader["option"], f'score {leader["positioning_score"]:.1f}')
    + metric("Margin over #2", f'+{margin:.1f}', f'vs {second["option"]}')
    + metric("Weakest evidence", weakest_evidence["option"], "least-evidenced framing")
    + "</div>"
)

Interpretation (provisional)

  • CRM Platform wins on clarity but is weak on differentiation — buyers get it instantly, yet see it as commoditised (cf. evidence EV-007).
  • Revenue Operations Platform balances differentiation, pain alignment, and scalability.
  • AI Readiness Platform scores highest on investor appeal but lowest on evidence strength — the most exciting story is the least evidenced today (cf. Hypothesis H-CRM-AI).
WarningThis is a framing tool, not a verdict

The scores are one analyst’s synthetic judgement. Positioning should be decided against customer evidence — message testing, win/loss analysis, discovery calls — not an internal scoring exercise. See Working Paper Ch. 4.
