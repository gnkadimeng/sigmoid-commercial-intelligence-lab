plus code in `src/` and the notebooks. If you cannot regenerate a file here, it does not belong here.

- Do not hand-edit processed files; change the code that produces them.
- Document the generating script/notebook for each artifact.
- Git-ignored by default; regenerate rather than commit large outputs.
